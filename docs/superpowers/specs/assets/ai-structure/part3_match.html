SCORE — CPEF, one number per directionBackend
  • Each direction = one side's preferences scored against the other side's attributes
  • Raw fit per signal: f = preferred value vs the matching attribute
  • Two-sided confidence: c = c_A × c_B → trust-gain ρ
program-side c by source: claimed school 1.0 > verified feed 0.85 > crawler 0.6 > inferred 0.4
  • Shrink: f̂ = ρ·f + (1−ρ)·prior — sure signals move it, unsure ones barely do
  • Weight: A = (importance/10)·ρ, averaged over present signals only
  • Deal-breakers: confidence-aware veto craters degree / visa / over-budget — no separate filter
  • Coverage damp: thin / guessed profile pulled toward neutral → deeper = sharper
CPEF = g · V · ( Σ A·f̂ ) / ( Σ A )  ∈ [0,1]
School rankings (US News / QS) are shown on cards, never scored — not a quant value the match considers.
