Unclaimed → we determine the whole profile

For an unclaimed program we set both sides from public materials: attributes (the facts above) and preferences — a baseline "target applicant" inferred from the public class profile (typical admitted GPA / test bands, common feeder fields). So program → student works before any claim. When the school claims, they overwrite both with their real data.
