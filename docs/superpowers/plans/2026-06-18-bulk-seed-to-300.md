# Bulk-seed to US-News top 300 — Implementation Plan

> **For agentic workers:** executed inline this session (orchestration-heavy: workflows + migrations). Steps use checkbox tracking.

**Goal:** Seed every US-News National Universities rank 1–300 (~260 net-new) at full treatment — verified stats + flagship programs + Commons campus photos — then refocus `enrich-profile` on enrichment+repair only.

**Architecture:** Deterministic core (College Scorecard bulk CSV → stats + programs + factual description, zero fabrication) + agent-discovered, Commons-API-verified photos. Tiered ~50/batch migrations mirroring `seed12univ1`. Spec: `docs/superpowers/specs/2026-06-18-bulk-seed-to-300-and-routine-refocus-design.md`.

**Tech:** Python (csv), College Scorecard bulk CSV, Wikimedia Commons API, Alembic data migrations, Workflow tool for photo discovery.

---

### Phase 0: Foundation (one-time)

- [ ] **0.1 Download Scorecard institution CSV** — `Most-Recent-Cohorts-Institution.csv` from the official key-free download; confirm it parses and has UNITID + the needed fields (INSTNM, CITY, STABBR, CONTROL, ADM_RATE, NPT4, MD_EARN_WNE_P10/_4YR, C150_4, UGDS, LATITUDE, LONGITUDE, CCBASIC). Cache to `/tmp/scorecard_inst.csv`.
- [ ] **0.2 Source US-News National Universities 1–300** — workflow: agents compile rank→official-name for ranks 1–300 from US-News + corroborating mirrors; flag low-confidence ranks. Save `/tmp/usnews_300.json`.
- [ ] **0.3 Match name→UNITID** — fuzzy-match each ranked name to a Scorecard row; auto-accept high-confidence, set aside ambiguous for manual verification; drop the ~40 already in the DB (query live API for existing names). Output `/tmp/seed_master.json` = the to-seed set with verified stats + rank.
- [ ] **0.4 Top-5 programs per UNITID** — from `Most-Recent-Cohorts-Field-of-Study.csv` (top CIPs by completions, credential ≥ bachelor's); attach to each master entry. Real field names only.

### Phase 1..N: Per-tier seed (repeat for rank tiers, ~50/batch)

For each tier (e.g., 34–100, 101–150, 151–200, 201–250, 251–300):

- [ ] **T.1 Photos** — photo-discovery workflow over the tier's universities → candidate Commons filenames; then deterministic Commons-API credit verification → `{url,credit}` (≤5 each; omit when none verify).
- [ ] **T.2 Build migration** — generator emits `seed300t<N>_<rank>.py` + sibling `seed300t<N>_data.json` (admin User + Institution + ranking_data + school_outcomes{stats,location,campus_photos} + 5 published programs; NO `_standard`). Idempotent skip-if-exists; downgrade deletes the batch. `down_revision` = live `origin/main` head (head-sync immediately before).
- [ ] **T.3 Validate** — ruff clean; scratch-DB `alembic upgrade head` from live head → assert tier institutions+programs exist with stats/photos; `test_alembic_has_single_head` passes; downgrade reverses; drop scratch.
- [ ] **T.4 Ship** — PR (`feat(profiles): seed US-News ranks <a>–<b>`); auto-merge on green → deploy. Watch for dual-head; if one appears, ship a merge migration.
- [ ] **T.5 Verify live** — fleet `total` climbed by tier size; spot-check 2 cards + 1 detail page render. Report count.
